Prerequisities
==============

Ubuntu
------
apt-get install nodejs npm texlive texlive-lang-czechslovak texlive-latex-extra xpdf

Fedora
------
dnf install texlive-adjustbox texlive-babel-czech xpdf

Common
------
npm install moment

Use
===
node generate.js
pdflatex output.tex
pdflatex output.tex
xpdf output.pdf
